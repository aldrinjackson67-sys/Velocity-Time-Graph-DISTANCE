# Velocity vs Time Graph Project

This Python project plots a Velocity vs Time graph and calculates distance travelled using numerical integration.

## Features

* Velocity vs Time graph plotting
* Distance calculation from graph
* Percentage error calculation
* Automatic graph image generation

## Technologies Used

* Python
* NumPy
* Matplotlib

## Formula Used

Distance is calculated using the area under the velocity-time graph.

## Output

* Velocity-Time Graph
* Distance Calculation
* Percentage Error
